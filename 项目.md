# 魔法自动化工具

做一个 chrome 插件, 主要功能是通过 ai 来自动化操作浏览器页面.

1. 用于自定义接入 openai 接口.
2. 通过对话的方式, 让用户输入想要在网页上完成的操作.
3. 插件会根据用户的输入, 生成相应的代码, 并在浏览器中执行, 自动化操作网页, 包括点击按钮, 填写表单, 滚动页面等.
4. 插件会保存代码, 以便下次用户可以直接执行对应的操作.
5. ai 的输入包含完整的 html 和用户输入的操作需求, 输出是可以直接执行的 js 代码.
6. ai 模型分成两类, 文本模型和视觉模型, 对于视觉模型, 需要发送当前网页的截图, 以便模型能够识别页面元素的位置和内容, 从而生成更准确的代码.
